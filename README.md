# Transcribe AI – Frontend

React frontend for Transcribe AI.
Provides a simple UI to upload files, transcribe audio/video, and translate into multiple languages.

---

## 🚀 Features

* File upload (audio/video)
* Language selection
* Real-time transcription display
* Translation output
* API integration with Django backend

---

## 🛠 Tech Stack

* React
* Fetch API
* CSS

---

## ⚙️ Setup

```bash
git clone https://github.com/your-username/transcribe-ai-frontend.git
cd transcribe-ai-frontend

npm install
```

---

## ▶️ Run App

```bash
npm start
```

---

## 🔗 Backend API

Make sure backend is running:

```
http://127.0.0.1:8000/api/transcribe/
```

---

## ⚠️ Notes

* Ensure CORS is enabled in backend
* Works with audio and video files
* Keep file size small for testing
